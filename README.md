# BrainfuckJS
Hide Text and Javascript inside < | > characters

Features:
 - Encrypt a string into Brainfuck similar content
 - Decrypt Brainfuck encrypted content back to original

Example:

 "Hello World"
 ```javascript
 ">><|<< >|<|> >|<|><< >|<|><< >|>|> >>|<< ><<|>>< >|>|> >|>|<> >|<|><< >|<|<"
```
